## 📂 Proyectos destacados

### 🤖 Robot Oruga Explorador

Robot explorador controlado mediante WiFi utilizando **ESP32**, capaz de transmitir video en tiempo real y ser controlado desde una interfaz web.

**Tecnologías:**
ESP32 · HTML · CSS · JavaScript · IoT

---

### 🌡️ Sistema IoT de Monitoreo Ambiental

Sistema de monitoreo ambiental que mide **temperatura y humedad en tiempo real** utilizando sensores conectados a un microcontrolador.
Los datos se muestran en una pantalla LCD y se registran automáticamente en una **tarjeta SD cada 10 minutos**, incluyendo fecha y hora mediante un módulo **RTC**.
El sistema también activa un ventilador automáticamente cuando la temperatura supera un umbral definido.

**Tecnologías:**
Arduino · Sensores ambientales · RTC · SD Card · IoT

---

### 🌱 AsisteAgro

Aplicación web que proporciona **información técnica sobre cultivos agrícolas**, incluyendo condiciones ideales, etapas de crecimiento, enfermedades comunes y recomendaciones.

**Tecnologías:**
React · Vite · Tailwind CSS · JavaScript

---

### 📅 Sistema de Agendamiento de Talentos

Aplicación web desarrollada para gestionar el **agendamiento de citas y espacios de trabajo**, permitiendo organizar horarios y optimizar la disponibilidad de recursos.

**Tecnologías:**
React · Vite · JavaScript

## ⚙️ Instalación y uso

Clonar el repositorio:

```bash
git clone https://github.com/Yulian64/portfolio.git
```

Entrar al proyecto:

```bash
cd portfolio
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npm run dev
```
